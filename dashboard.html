University Technology MARA

The UiTM Students e-Profile website is a modern platform that allows users to view and explore the academic and extracurricular profiles of UiTM students. Featuring a user-friendly interface, it includes a secure login, a list of student profiles with detailed information, and individual profile pages. The site is designed with a responsive layout using Bootstrap and offers a dark mode toggle for a customizable viewing experience. It provides an efficient way to showcase student achievements and information in a clean, accessible format.
